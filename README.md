# Smart Path Finder

## Description

This project helps visually impaired people navigate safely using Arduino.

## Components Used

* Arduino Nano
* Ultrasonic Sensor(HC-SR04)
* Vibration Motor
* Rechargeable Li-ion Battery

## Working

The ultrasonic sensor detects obstacles. If an object is close, the vibration Motor alerts the user.

## How to Run

1. Open Arduino IDE
2. Upload the code
3. Connect the components
4. Power ON
