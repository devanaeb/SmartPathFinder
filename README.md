# Smart Path Finder

## Description

Smart Path Finder is a compact assistive navigation system designed for visually impaired individuals. It uses an ultrasonic sensor to detect obstacles in real time and provides intuitive feedback through a vibration motor.

Built using an Arduino Nano and powered by a rechargeable Li-ion battery, the system is portable, energy-efficient, and suitable for everyday use. The project focuses on enhancing user safety and independence through a simple, low-cost solution.


## Components Used

* Arduino Nano
* Ultrasonic Sensor(HC-SR04)
* Vibration Motor
* Rechargeable Li-ion Battery

## Working

The ultrasonic sensor continuously measures the distance between the user and surrounding obstacles.

* When an object is detected within a predefined range,
* The Arduino Nano processes the data
* The vibration motor is activated

The vibration intensity or pattern can indicate how close the obstacle is, helping the user navigate safely.

## How to Run

1. Open Arduino IDE
2. Upload the code to Arduino Nano using Arduino IDE
3. Connect the ultrasonic sensor and vibration motor properly
4. Power the system using the Li-ion battery
5. Hold or attach the device while walking

## Features

* Real-time obstacle detection
* Silent feedback using vibration (ideal for noisy environments)
* Compact and portable design
* Low power consumption
